Here is the official link for installing Laravel. But instead of following the "official" standard way, we will follow a more manageable and easy way. Start with installing the products from the left side table, in that order. Laragon is a Vagrant Box for windows. It provides you with an already build server. Now install PhpMyAdmin. Just download it and extract to {LARAGON_DIR}\etc\apps\phpMyAdmin. Koala is a compiler for SASS/SCSS etc. Useful! PhpStorm is the best ide out there for laravel development. But it is paid, has educational free versions which you need to renew every year. Node.js and composer are package manager for js and dependency manager for PHP respectively. After downloading them, you may have to add composer and npm to environment variables. You can google how to do this. All of the above installations are one-time setup. Once done correctly, forget about it!
Now for installing Laravel, copy following line of code in command prompt:

composer global require "laravel/installer"

This installs "laravel" on your PC. After this you have to make a project in laravel, go to the directory in which you want to make laravel project. Run the following command in the console:

laravel new PROJECT_NAME

After some time, this will show installation complete. Now let me explain

Recommended Packages

  • Voyager
  • Font-Awesome
  • Vue
  • Vue Router

These packages are good for starting new projects as they provide kickstart. Voyager is used as admin panel and offers basic curd. Vue is frontend framework, wonderful one! Font-awesome is one of the best fonts pack available out there. Also, I have not mentioned bootstrap, jquery as they are already integrated with Laravel. Below I have explained the procedure of installing these packages in the project.

Installing Voyager

Installing voyager is pretty straight. Voyager uses storage and database, so it would be useful to setup them before installing voyager. Also, I usually install voyager right after creating the new project. So installing other required packages before installing voyager would also be a good idea. Here is the code to install other required packages, setting up storage and connecting to the database.

npm install
php artisan storage:link

In .env file, edit database connections and change APP_URL to URL of the domain (with port)

APP_URL= http://localhost:8000

This is necessary for proper functioning of voyager. Well, we are completed with the pre-requisites. Now, for installing Voyager, you can follow the Official Guide. Here are the basic commands needed to install and setup voyager.

composer require tcg/voyager
php artisan voyager:install
php aritsan voyager:admin --create

Installing Font-Awesome

Font Awesome has a very large community and it has a large database of icons. For finding the code for different icons, you can visit Font Awesome Official Site. On the same page, you will find link to getting started with font-awesome, thou there is nothing more than copy pasting. Enough with the talk, for installing font-awesome use this code:

npm install
npm install font-awesome

Note that, for installing any javascript resource like font-awesome, the code is pretty much the same. Anyways, after doing this, add following line of code in resources/assets/sass/app.sass:

@import "~font-awesome/scss/font-awesome";

Installing Vue and Vue Router

Installing Vue and Vue Router is similar to that of font-awesome. Just type:

npm install vue
npm install vue-router

Vue and Vue-Router require certain configurations. All of which happens in js/app.js and sass/app.sass file of resources/assets directory. Copy the following code and replace it in app.js file:

require('./bootstrap');
window.Vue = require('vue');
import Vue from 'vue'
import VueRouter from 'vue-router'
Vue.use(VueRouter);

const app = new Vue({
el: '#app',
});

Voila! We are ready to go. Go to console, type

npm watch run

Open the laragon page for that project, i.e. {PROJECT NAME}.oo, if you have not changed the settings. Now go to {PROJECT NAME}.oo/admin to see voyager working. Go to localhost/phpmyadmin to see the database.
